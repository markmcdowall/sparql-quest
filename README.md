# Requirements
- Python 3.6+
- Python Modules:
  - pylint
  - pytest
  - requests
  - python-dateutil
  - sphinx

Installation
------------

Directly from GitHub:

```
git clone https://github.com/markmcdowall/sparql_quest.git

cd sparl_quest
```

Create the Python environment

```
pip install -e .
pip install -r requirements.txt
```

Running the Tests
-----------------

```
pytest
```
